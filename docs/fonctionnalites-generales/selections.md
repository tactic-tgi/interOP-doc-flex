À travers le logiciel, vous aurez des menus déroulants de sélection vous permettant d'avoir une constance dans l'inscription des données. Leur contenu peut être :

- Choix fixe (ex: langue)
- Table du logiciel (ex: les clients actifs)
- Listes de valeurs écrites dans les paramètres système (ex: catégories de clients)

## Effacer {#effacer}

Lorsque vous désirez effacer le choix d'un menu déroulant, appuyez sur le petit X se trouvant à la droite du menu.

![](/img/Fonctionnalites_Menus_1.png)

## Sortir d'un menu déroulant {#sortir}

Lorsque vous désirez sortir d'un menu déroulant sans faire de sélection, vous pouvez cliquer dans l'entête de la fenêtre ou bien faire le bouton `Escape` sur votre clavier. 

![](/img/MenuDeroulant_sortir.gif)

## Filtres {#filtres}

Certains menus de sélection peuvent ne pas afficher l'entièreté de la table visée, selon le contexte donné. Le logiciel noyau contient des filtres de base qui permettent de ne pas faire de mauvais choix. Vous pouvez demander l'ajout de filtres à certains endroits si vous le désirez.

Un filtre de base s'applique automatiquement dans tous les contextes ; il s'agit du fait que l'enregistrement est actif (seulement s'il peut être désactivé).

## Grouper les données de la liste {#grouper}
Dans les filtres, si vous décidez de grouper les données, vous pourrez alors manipuler l'ouverture des regroupements dans les listes.
Vous pourrez ouvrir et refermer des sections, pour mieux consulter les données dont vous avez besoin.
![](/img/multi_selection_grouper.png)

Pour tout refermer ou réouvrir une section d'un seul coup, tenir la touche Majuscule en cliquant sur la ligne. 
![](/img/multi_selection_shift.gif)

## Sélection de cases à cocher {#cases-cocher}
Dans certaines listes, comme les réquisitions, vous pourrez sélectionner des items à commander en cochant des cases au bout de la ligne.
Il y a 3 façons de faire : 

  1- Sélectionner manuellement chaque item en cliquant sur chaque case.

  2- Sélectionner la case située dans l'entête bleue du fournisseur (ou selon le regroupement choisit), pour sélectionner tous les produits associés à celui-ci.

  3- Sélectionner une case, puis tenir la touche Majuscule enfoncée en sélectionnant une case plus bas pour sélectionner tous les items entre les deux.

  ![](/img/multi_selection_caseCocher.gif)

## Recherche {#recherche}

Lorsque nous jugeons que la sélection pourra dépasser un certain nombre de choix, nous ajoutons une barre de recherche pour trouver facilement un enregistrement précis. Si un cas n'avais pas été prévu, vous pouvez demander d'en ajouter une facilement.
